A lightweight, fast, and privacy-focused web viewer for your Star Citizen hangar and buyback inventory.

---

## 🇺🇸 English Description

**Hangar Viewer** is a Single Page Application (SPA) designed to help Star Citizen players analyze their RSI inventory without uploading data to external servers. It provides a clean interface to track fleet value, filter items, and detect special insurance statuses.

### ✨ Features
- **Total Fleet Value**: Automatic calculation based on your JSON data.
- **Dual View Modes**: Switch between a visual **Grid View** and a data-dense **List View**.
- **Smart Filters**: Filter by name or category (Ships, Flair, Skins, etc.).
- **Context Switching**: Easily toggle between your **Main Hangar** and **Buy Back** list.
- **LTI Detector**: Automatically scans sub-items to identify **Lifetime Insurance**.
- **Privacy First**: 100% client-side processing. Your JSON data never leaves your browser.

### 🚀 How to use
1. Export your hangar data using the [GuildSwarm - Hangar Manager Extension](https://chromewebstore.google.com/detail/guildswarm-hangar-manager/bfjobjficjfnhknonfpnndcmgmgiidkp).
2. Open `viewer.html` in any modern web browser.
3. Click on **"Open JSON"** and select your exported file.
4. *Pro Tip*: Rename your file to `myHangar.json` in the same folder to enable automatic loading.

### 🛠️ Tech Stack
- Vanilla JavaScript (ES6+)
- Tailwind CSS
- HTML5

## 🇪🇸 Descripción en Español

**Hangar Viewer** es una aplicación web de una sola página (SPA) diseñada para ayudar a los jugadores de Star Citizen a analizar su inventario de RSI sin necesidad de subir sus datos a servidores externos. Ofrece una interfaz limpia para rastrear el valor de la flota, filtrar ítems y detectar estados de seguro especiales.

### ✨ Características
- **Valor Total de la Flota**: Cálculo automático basado en tus datos JSON.
- **Modos de Vista Dual**: Cambia entre una **Vista de Cuadrícula** visual y una **Vista de Lista** densa en datos.
- **Filtros Inteligentes**: Busca por nombre o filtra por categorías (Naves, Flair, Skins, etc.).
- **Cambio de Contexto**: Alterna fácilmente entre tu **Hangar Principal** y tu lista de **Buy Back**.
- **Detector de LTI**: Escaneo automático de sub-ítems para identificar seguros de por vida (**Lifetime Insurance**).
- **Privacidad Total**: Procesamiento 100% en el cliente. Tus datos nunca salen de tu navegador.

### 🚀 Instrucciones de uso
1. Exporta los datos de tu hangar usando la extensión GuildSwarm - Hangar Manager.
2. Abre el archivo `viewer.html` en cualquier navegador moderno.
3. Haz clic en **"Abrir JSON"** y selecciona tu archivo exportado.
4. *Consejo*: Renombra tu archivo a `myHangar.json` en la misma carpeta para permitir la carga automática.
